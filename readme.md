# EMOJI_LOOKUP

You can type "sad" to find the related "sad" emojis

## HOW TO RUN

you can clone this repository by typing

### 1. clone this repo

```
git clone git@github.com:sasakiRoo/emoji_lookup.git
```

### 2. install packages

```
npm install
```

### 3. run

```
node index.js
```
